Alphacast is a financial data analytics platform that connects you to economic and market data from around the world. Use the REST API to manage datasets, build transformation pipelines, and pull data from providers like FRED, the World Bank, BLS, Yahoo Finance, and more — all from a single, unified interface.

## Get started in three steps

<Steps>
  <Step title="Get your API key">
    Log in to [alphacast.io](https://alphacast.io) and copy your API key from your account settings. You will pass this key on every API call.
  </Step>
  <Step title="Create a repository">
    Repositories are top-level containers for your datasets and pipelines. Create one via the API or the web app before you start uploading data.

    ```bash
    curl -X POST https://api.alphacast.io/repositories \
      -u YOUR_API_KEY: \
      -H "Content-Type: application/json" \
      -d '{"name": "My First Repository", "description": "Getting started", "privacy": "Private"}'
    ```
  </Step>
  <Step title="Fetch data from a provider">
    Browse 15+ built-in data providers — FRED, BLS, World Bank, Yahoo Finance, OECD, and more — and pull series data directly into your pipelines or export to CSV/JSON.

    ```bash
    curl "https://api.alphacast.io/providers" \
      -u YOUR_API_KEY:
    ```
  </Step>
</Steps>
