## Browse by topic

The World Bank organizes indicators into topic groups (Agriculture, Economy & Growth, Education, Health, etc.). Start at the root to see all topics.

```bash
# List all topics
curl https://api.alphacast.io/providers/worldbank/browse
```

**Example response:**

```json
{
  "type": "folder_list",
  "items": [
    {"id": "1", "name": "Agriculture & Rural Development", "type": "folder"},
    {"id": "3", "name": "Economy & Growth", "type": "folder"},
    {"id": "4", "name": "Education", "type": "folder"},
    {"id": "8", "name": "Health", "type": "folder"}
  ],
  "breadcrumbs": []
}
```

Navigate into a topic by passing its `id` as the `path` parameter.

```bash
# Browse indicators in "Economy & Growth" (topic ID 3)
curl "https://api.alphacast.io/providers/worldbank/browse?path=3"
```

The response lists all indicators in that topic as a `series_list`, each with an `id` (the indicator code) and `label`.

## Preview a series

Retrieve metadata and the last 24 data points to confirm you have the right indicator.

```bash
curl https://api.alphacast.io/providers/worldbank/series/NY.GDP.MKTP.CD
```

**Example response:**

```json
{
  "series_id": "NY.GDP.MKTP.CD",
  "label": "GDP (current US$)",
  "is_preview": true,
  "metadata": {
    "indicator_name": "GDP (current US$)",
    "country": "all",
    "total_observations": 24,
    "startyear": "2000",
    "endyear": "2023"
  },
  "data": [
    {"date": "2000-01-01", "value": 33618762424.0},
    {"date": "2001-01-01", "value": 34156090327.0}
  ]
}
```

<Note>
  Annual World Bank observations are returned with a date of `YYYY-01-01` (January 1 of the reference year). The `date` field in the response is always a full ISO date string.
</Note>

## Fetch the full series

Use POST to retrieve the complete historical record. Pass optional parameters in the `params` object to filter by country or date range.

<CodeGroup>

```bash All countries, full history
curl -X POST https://api.alphacast.io/providers/worldbank/series/NY.GDP.MKTP.CD/data \
  -H "Content-Type: application/json" \
  -d '{}'
```

```bash Single country (Brazil)
curl -X POST https://api.alphacast.io/providers/worldbank/series/NY.GDP.MKTP.CD/data \
  -H "Content-Type: application/json" \
  -d '{"params": {"country": "BR"}}'
```

```bash Specific date range
curl -X POST https://api.alphacast.io/providers/worldbank/series/NY.GDP.PCAP.CD/data \
  -H "Content-Type: application/json" \
  -d '{"params": {"country": "US", "date_range": "2000:2023"}}'
```

</CodeGroup>

### Supported parameters

| Parameter | Values | Default | Description |
|---|---|---|---|
| `country` | ISO 2-letter code or `all` | `all` | Filter by country. Examples: `US`, `CN`, `BR`, `AR`, `DE`, `IN`. |
| `date_range` | `YYYY:YYYY` | `1960:2026` | Limit observations to a year range. |

The following country codes are supported in the `country` parameter: `all`, `US`, `CN`, `JP`, `DE`, `GB`, `FR`, `IN`, `BR`, `MX`, `AR`, `CA`, `AU`, `KR`, `IT`, `ES`, `CL`, `CO`, `ZA`, `NG`.

## Complete workflow: browse, search, and fetch

<Steps>
  <Step title="Browse topics to discover indicators">
    ```bash
    curl https://api.alphacast.io/providers/worldbank/browse
    ```

    Identify the topic ID that contains the indicators you need.
  </Step>

  <Step title="Navigate into a topic">
    ```bash
    # Economy & Growth (topic 3)
    curl "https://api.alphacast.io/providers/worldbank/browse?path=3"
    ```

    Review the returned `series_list` for relevant indicator IDs.
  </Step>

  <Step title="Preview a specific indicator">
    ```bash
    curl https://api.alphacast.io/providers/worldbank/series/NY.GDP.MKTP.KD.ZG
    ```

    Confirm the label reads "GDP Growth (annual %)" and check the date range in `metadata`.
  </Step>

  <Step title="Fetch GDP growth for all countries">
    ```bash
    curl -X POST \
      https://api.alphacast.io/providers/worldbank/series/NY.GDP.MKTP.KD.ZG/data \
      -H "Content-Type: application/json" \
      -d '{"params": {"date_range": "2000:2023"}}'
    ```

    The response includes one observation per country per year. When `country` is `all`, each `data` entry includes the country identifier in `label`.
  </Step>

  <Step title="Filter to a single country">
    ```bash
    curl -X POST \
      https://api.alphacast.io/providers/worldbank/series/NY.GDP.MKTP.KD.ZG/data \
      -H "Content-Type: application/json" \
      -d '{"params": {"country": "BR", "date_range": "2010:2023"}}'
    ```

    When a specific country is selected, `label` in the response reads `"GDP Growth (annual %) â€” Brazil"`.
  </Step>
</Steps>

  <Accordion title="Understanding World Bank indicator IDs">
    World Bank indicator codes follow a dot-separated topic hierarchy. For example, `NY.GDP.MKTP.CD` breaks down as:

    - `NY` â€” National Accounts
    - `GDP` â€” Gross Domestic Product
    - `MKTP` â€” Market prices, total
    - `CD` â€” Current USD

    You do not need to memorize this structure â€” browsing by topic or using direct search is the easier path to finding the right indicator.
  </Accordion>

  <Accordion title="Data availability and coverage">
    Coverage varies by indicator and country. High-income countries typically have data back to 1960; lower-income countries may have shorter or more gaps-heavy series. The World Bank updates most indicators annually, usually with a 1â€“2 year publication lag. Preview a series before relying on its most recent values.
  </Accordion>
