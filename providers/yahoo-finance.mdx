title: "Fetch Stock Market Data from Yahoo Finance"

The Yahoo Finance integration in Alphacast gives you equity, ETF, and index price history for thousands of global instruments. You can search for tickers by company name or symbol, retrieve instrument metadata, and download historical OHLCV (open, high, low, close, volume) data in multiple formats. No API key or Yahoo Finance account is required.

## Endpoints

All Yahoo Finance endpoints live under `/tickers/`.

| Method | Path | Description |
|---|---|---|
| `GET` | `/tickers/search/{query}` | Search for tickers by name or symbol |
| `GET` | `/tickers/{ticker}` | Get metadata and summary for a ticker |
| `GET` | `/tickers/{ticker}/data` | Download historical price data |

## Search for tickers

Search by company name, ticker symbol, or partial string. The response returns a list of matching instruments with their symbols and names.

```bash
curl https://api.alphacast.io/tickers/search/apple
```

**Example response:**

```json
{
  "results": [
    {"symbol": "AAPL", "name": "Apple Inc.", "exchange": "NMS", "type": "EQUITY"},
    {"symbol": "AAPL.BA", "name": "Apple Inc.", "exchange": "BUE", "type": "EQUITY"}
  ]
}
```

Use the `symbol` value from search results as the `{ticker}` parameter in the detail and data endpoints.

## Get ticker detail

Retrieve metadata and a summary for a specific instrument, including the current price, market cap, sector, and description.

```bash
curl https://api.alphacast.io/tickers/AAPL
```

## Download price history

Fetch historical OHLCV data for a ticker. The default output is CSV and the default period is the last 5 years.

```bash
# Default: CSV, last 5 years
curl https://api.alphacast.io/tickers/AAPL/data

# JSON format, 1-year period
curl "https://api.alphacast.io/tickers/AAPL/data?\$format=json&period=1y"

# XLSX download, maximum available history
curl "https://api.alphacast.io/tickers/AAPL/data?\$format=xlsx&period=max" -o aapl.xlsx

# Most recent 20 trading days as TSV
curl "https://api.alphacast.io/tickers/AAPL/data?\$format=tsv&\$top=20"
```

### Query parameters

| Parameter | Values | Default | Description |
|---|---|---|---|
| `$format` | `CSV`, `JSON`, `XLSX`, `TSV` | `CSV` | Output format for the response |
| `$top` | integer | 0 (all) | Return only the N most recent data rows |
| `period` | `1d`, `5d`, `1mo`, `3mo`, `6mo`, `1y`, `2y`, `5y`, `10y`, `ytd`, `max` | `5y` | Time period to fetch |

<Warning>
  The `period` parameter uses Yahoo Finance period codes, not calendar dates. If you need data starting from a specific date, use `period=max` and filter the result in your pipeline.
</Warning>

## Example: AAPL price history

<Steps>
  <Step title="Search for the ticker">
    Confirm the exact symbol before fetching data, especially for non-U.S. listings.

    ```bash
    curl https://api.alphacast.io/tickers/search/apple
    ```
  </Step>

  <Step title="Preview ticker metadata">
    ```bash
    curl https://api.alphacast.io/tickers/AAPL
    ```

  </Step>

  <Step title="Download 2 years of daily closes as JSON">

  <Step title="Or download as Excel">

    ```bash
      -o aapl_5y.xlsx
    ```
  </Step>
</Steps>

## Common tickers

| Ticker | Instrument |
|---|---|
| `AAPL` | Apple Inc. |
| `MSFT` | Microsoft Corporation |
| `GOOGL` | Alphabet Inc. (Class A) |
| `SPY` | SPDR S&P 500 ETF Trust |
| `QQQ` | Invesco QQQ Trust (Nasdaq-100) |
| `^GSPC` | S&P 500 Index |
| `^DJI` | Dow Jones Industrial Average |
| `GLD` | SPDR Gold Shares ETF |
| `BTC-USD` | Bitcoin / USD |
| `EUR=X` | EUR/USD exchange rate |

<Tip>
  Yahoo Finance uses `^` as a prefix for indices (e.g., `^GSPC`, `^DJI`, `^VIX`) and `-USD` or `=X` suffixes for crypto and forex pairs. These follow the same format in Alphacast.
</Tip>

## Using Yahoo Finance in pipelines

To pull ticker data into an Alphacast pipeline, add a `fetch-yahoo` step.

```json
{
  "type": "fetch-yahoo",
  "order": 0,
  "configuration": {
    "tickers": ["AAPL"],
    "period": "5y"
  }
}
```

The step produces a dataset with columns for date, open, high, low, close, volume, and adjusted close, ready for further transformation steps in the pipeline.

<Note>
  Yahoo Finance data is sourced from Yahoo's public market data feeds. Alphacast does not control data availability, coverage, or accuracy. For production-grade financial data requiring guaranteed uptime or licensing, consider a commercial data provider.
</Note>
