The Alphacast Model Context Protocol (MCP) server lets any MCP-compatible client interact with the Alphacast platform through structured tool calls. Search the curated catalog, download dataset rows, browse 15+ external data providers, and build or edit data pipelines — all from inside Claude Desktop, Claude Code, Cursor, or any other client that speaks MCP.

## Server endpoint

<Note>
  **Public endpoint:** `https://mcp.alphacast.io` (HTTP streamable transport). Authenticate with the same API key you use for the REST API, sent as a Bearer token in the `Authorization` header.
</Note>

There is nothing to install. Point your client at the URL above, paste your API key, and the 22 tools below become available immediately.

## Safety boundaries

The server intentionally does **not** expose destructive operations. Repositories, datasets, and entire pipelines can only be deleted from the Alphacast web UI — never from an agent. Pipeline steps can be added, edited, and removed, since each mutation is fully reversible from the Alphacast web UI.
